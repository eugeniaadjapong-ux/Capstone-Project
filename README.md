# Weather Forecast Web Application

## Project Description

This project is a weather forecast web application that allows users to search for a city and retrieve real-time weather data using a third-party API. The application provides current weather conditions, temperature, weather icons, and basic forecast information.

The application is built as a capstone project to demonstrate basic web development and API integration skills.

## Features

* Search for city weather information
* Display current temperature
* Show weather condition (sunny, cloudy, rainy, etc.)
* Display weather icons
* Provide daily weather forecast
* Handle invalid city search errors

## Technologies Used

* HTML
* CSS
* JavaScript (or your chosen framework if applicable)
* API Integration

## API Used

This project uses the OpenWeatherMap API to fetch real-time weather data.

## How to Run the Project Locally

1. Clone the repository from GitHub.
2. Open the project folder.
3. Open the `index.html` file in a browser (if using plain HTML/CSS/JavaScript).
4. Ensure internet connection for API requests.
